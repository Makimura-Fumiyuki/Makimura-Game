# 牧村ゲームハブ プロジェクト

「牧村が作ったゲーム集」(makimura-gamehub.com) の開発。運営者の牧村さんは非エンジニア。

> **このファイルは生きた引き継ぎ書。** 新しい約束事・こだわり・落とし穴が判明したら、その都度ここに追記すること（牧村さんからの常設指示）。

## プロジェクト構成

- 全ゲーム1ファイル完結のHTML/CSS/JS。ビルドなし、フレームワークなし、画像ファイルなし（サムネはSVGインライン描画）
- 音はWeb Audioで全合成
- バックエンド: Supabase（`scores`テーブル + Realtime。接続情報はdox.html内のものを全体で流用。公開キーはINSERT/SELECTのみ許可済み）
- ホスティング: Cloudflare Pages（GitHubリポジトリ `Makimura-Fumiyuki/Makimura-Game` 連携）
- デプロイ: 牧村さんがGitHub Web UIへファイルをドラッグ&ドロップ（git CLIは使わない。コミットやプッシュを促さない）
